Building Tomorrow's AI leaders in Tanzania

Jul 22, 2018 · last updated on Jul 27, 2018

A detailed weekly program for a six weeks industrial training in machine learning and data science for university students in Tanzania.

Introduction

Pythontz in collaboration with parrotai is hosting a six weeks intesnive training in machine intelligence for students. The training is designed to help students grasp both theoretical and practical foundations of machine learning. A detailed weekly program for this training is as follows:

Week 1: Machine learning fundamentals

  1. Basics of Python for ML and DS Project.
    • Python for ML and DS
    • An introduction to Objects and their Classes in python
  2. Selected MOOC & online courses.
    • Foundation of Machine Learning by sambaiga
    • ML by Andrew Ng week 1, 2 and 3
    • Data-driven Decision Making
  3. Book chapter.
    • Hands-on Machine Learning with Scikit-Learn and TensorFlow chapter 1,2,3 and 4
    • Introduction to Machine Learning with Python A Guide for Data Scientists chapter 1, 2 and 4
    • Deep Learning with Python by FRANÇOIS CHOLLET chapter 1 and 4
  4. Lab work

Week 2: Deep learning and Computer Vision

  1. Foundations of Deep learning
    • Foundation of Deep learning by sambaiga
    • Improving Deep Neural Networks: Hyperparameter tuning, Regularization and Optimization
    • CS231n: Training Neural Networks, part I and II
  2. Deep learning frameworks
    • Applied AI with DeepLearning week 3
    • CS231n: Deep Learning Hardware and Software
  3. Computer vision (CNN)
    • Deep learning for computer vision by sambaiga
    • Deep Learning: Do-It-Yourself (lecture 6&7)
    • Deep learning specialization Andrew Ng: CNN
    • Deep Learning in Computer Vision (focus week 3,4 and 5)
  4. Lab work
    • IndabaX Tanzania Lab 4

Week 3: Sequence model and Recommender systems

  1. Sequence Models and NLP
    • Deep Learning: Do-It-Yourself (lecture 8)
    • CMSC 35246 Deep Learning (lecture 6)
    • Deep learning specialization Andrew Ng: Sequence model
  2. Recommender systems
    • Introduction to Recommender Systems
  3. Building machine learning project
    • Structuring Machine Learning Projects
  4. Lab work

Week 4: Practical machine learning 1

Choose two tracks: track 1 is mandatory.

  1. Winning Kaggle challenge
    • How to Win a Data Science Competition: Learn from Top Kagglers
  2. Computer vision
    • Fast ai lessons 8, 9 and 10: Hiromi Suenaga notes
  3. Sequence model and recommender systems
    • Fast ai lesson 4, 5, 6: Hiromi Suenaga notes
    • Web Traffic Time Series Forecasting

Week 5: Practical machine learning II

Choose two tracks: track 1 is mandatory.

  1. General machine learning
    • Santander Customer Satisfaction: Which customers are happy customers?
    • TalkingData AdTracking Fraud Detection Challenge: Can you detect fraudulent click traffic for mobile app ads?
  2. Computer vision
    • Fast ai lesson 8, 9 and 10: Hiromi Suenaga notes
  3. Sequence models
    • Fast ai lesson 10 and 11: Hiromi Suenaga notes

Week 6: Basics of probabilistic methods for machine learning

  1. Selected MOOC and online courses
    • Bayesian Methods for Machine Learning (week 1 and 2)
    • Lecture 18: Learning Probabilistic Models
    • Basics of Probability and Information Theory
    • Bayesian Inference by sambaiga
  2. Book chapter
    • Bayesian Data Analysis by Andrew Gelman chapter 1
    • Bayesian Methods for Hackers chapter 1
